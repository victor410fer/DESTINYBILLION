# 🔒 Security Testing Repository

Welcome to my security testing repository! This project contains scripts and tools for security testing, created from Termux on Android.

## 👤 About
- **GitHub User**: [lonelyk757-art](https://github.com/lonelyk757-art)
- **Created**: January 2025
- **Platform**: Termux on Android
- **Status**: Active development

## 📁 Files Included
| File | Description |
|------|-------------|
| `security_test.sh` | Basic security test script |
| `security_audit.sh` | Comprehensive security audit |
| `github_test.sh` | GitHub connection test |
| `scan.py` | Python security scanner |
| `.github/workflows/security.yml` | GitHub Actions automation |

## 🚀 Quick Start
```bash
# Clone the repository
git clone https://github.com/lonelyk757-art/security-test.git

# Enter the project directory
cd security-test

# Run security audit
./security_audit.sh

# Run Python scanner
python scan.py

# Test GitHub connection
./github_test.sh
```

## 🤖 GitHub Actions

This repository uses GitHub Actions for automated testing. The workflow:

· Runs on every push to main branch
· Runs on pull requests
· Can be triggered manually

View Actions: https://github.com/lonelyk757-art/security-test/actions
